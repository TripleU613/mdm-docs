# Welcome + terms

Where it lives:
- `app/src/main/java/com/tripleu/ui/activities/WelcomePage.kt`

What it does:
- Shows the welcome screen and requires agreement before continuing.
- Links to the privacy policy.

How it runs:
- User must check the agreement checkbox to enable Continue.
- Tapping the terms text opens `PrivacyPolicyActivity`.
- Continue navigates to `login_page`.
