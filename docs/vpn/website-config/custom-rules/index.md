# Custom Rules

The **Custom Rules** tab allows manual rule creation.

You can define:
- Domain or subdomain
- Exact domain match
- Action:
  - Allow
  - Block
- Optional overrides:
  - Allow images for this rule
  - Allow words for this rule

This is useful for exceptions or highly specific access control.
