# StorePad

**StorePad** on the website is a direct copy of the **Android StorePad experience**. If you are already familiar with the Android version, this section should feel identical in behavior and purpose.

No new logic or workflows are introduced here.

---

## Overview

StorePad provides a centralized view of app availability, requests, and updates, mirroring the Android implementation.

## Available Controls

### Show on Lock Screen

- Option to display StorePad on the device lock screen
- Behavior matches the Android version

---

### App Catalog Search

- Search the app catalog by **keyword**
- Results and behavior are the same as on Android

---

### App Requests

- View apps requested by users
- Approve requested apps directly from the website
- Approval flow matches the Android StorePad behavior

---

### Muted Updates

- View **muted app updates**
- Muted updates are created by:
  - Swiping left on an update on the Android system
- You can also manually input muted updates from the website

Available actions:
- Approve muted updates
- Reject muted updates

---

### Available Apps Management

For available apps, you can:

- Approve apps
- Disapprove apps
- Reject apps

All actions behave exactly like the Android version.

---

## Summary

StorePad on the website is intentionally simple:

- Same layout
- Same controls
- Same behavior as Android

No additional explanation is required beyond the Android StorePad documentation.
