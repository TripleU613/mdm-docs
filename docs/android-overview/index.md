# Basic Android App Overview

## Home Screen Overview

After completing account setup and entering your **App PIN** on the authentication screen, the app opens to the **Home** page.

The Home page provides a high-level overview of the device and its current MDM state.

<details>
<summary>image</summary>

![Home screen after PIN authentication](/assets/home-screen.png){ width=320 }

</details>

### Home Page Sections

The Home page includes the following elements:

- **Applied Policies Count**  
  Displays how many policies are currently applied to the device.

- **Tech Tip of the Day**  
  A rotating informational tip.  
  > Note: This feature may not always display content, depending on available AI tokens.

- **Quick Facts**  
  General information about the MDM product.  
  > Some of this information may be outdated.

- **Contact Support**  
  Allows you to send a support message directly from the device.  
  Messages include:
  - Device identifier
  - Device-specific context  
  This ensures support can identify exactly which device is contacting them.

<details>
<summary>image</summary>

  ![Contact support form](/assets/contact-support.png){ width=320 }

</details>

- **Policy Status Overview**  
  A quick summary showing:
  - Which policies are currently applied
  - Which policies are not applied  

This page is intended as a **status dashboard**, giving you a quick view of what is active and what is not.

## Navigation Sidebar

From the Home screen, you can open the **sidebar navigation menu** to access all major sections of the app.

![Sidebar navigation menu](/assets/sidebar-nav.png){ width=320 }

### Available Tabs

The sidebar contains the following tabs:

1. **Home**  
   The main dashboard (current screen).

2. **Lock Screen**  
   The primary access gate for MDM. See the detailed overview below.

3. **System**  
   System-related controls and information. See the detailed overview below.

4. **Installation**  
   Installation-related settings and workflows. See the detailed overview below.

5. **Accessibility**  
   Accessibility-related configuration. See the detailed overview below.

6. **Network**  
   Network and connectivity management. See the detailed overview below.

7. **Store**  
   App storeâ€“related features. See the detailed overview below.

8. **Apps**  
   Application management. See the detailed overview below.

9. **Settings**  
   General app and account settings. See the detailed overview below.

Each tab provides access to a specific category of device management features.
