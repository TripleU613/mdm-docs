# TripleUMDM Docs

Documentation site for TripleUMDM.

## Contributing

- Edit or add pages inside the `docs/` folder.
- Keep filenames short and lowercase.
- Use the existing sections and tone.
- Add screenshots to `docs/public/assets/` and link them as `/assets/...`.
- For images, wrap them in a collapsible block:

```html
<details>
<summary>image</summary>

![Alt text](/assets/example.png){ width=520 }

</details>
```
